# 🎓 LectureLens-AI

LectureLens-AI is an intelligent AI-powered platform designed to enhance learning from lectures by transforming raw educational content into structured, digestible insights. It helps students and professionals extract key concepts, summaries, and actionable knowledge from lectures efficiently.

---

## 🚀 Features

* 🎧 **Lecture Processing**
  Upload or input lecture audio/video/text and let AI analyze it.

* 🧠 **Smart Summarization**
  Generates concise and structured summaries of long lectures.

* 📝 **Key Point Extraction**
  Identifies important concepts, highlights, and takeaways.

* ❓ **Question Generation**
  Automatically creates questions for revision and self-assessment.

* 🔍 **Searchable Insights**
  Easily find specific topics or concepts within lecture content.

* 📊 **Organized Learning**
  Converts unstructured lectures into structured notes.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/ShubhamSinghRawat10/LectureLens-AI.git
cd LectureLens-AI
```

### 2. Install dependencies

```bash
# Backend
cd backend
npm install   # or pip install -r requirements.txt

# Frontend
cd ../frontend
npm install
```

### 3. Run the application

```bash
# Backend
npm start   # or python app.py

# Frontend
npm run dev
```
